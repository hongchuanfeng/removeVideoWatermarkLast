# 快速启动指南

## 第一步：安装依赖

```bash
npm install
```

## 第二步：启动开发服务器

```bash
npm run dev
```

## 第三步：访问网站

打开浏览器访问：http://localhost:3000

网站会自动重定向到默认语言（英文）：http://localhost:3000/en

## 测试多语言

- 英文版本：http://localhost:3000/en
- 中文版本：http://localhost:3000/zh

## 生成 Favicon

1. 准备一个 32x32 像素的图标图片
2. 使用在线工具转换为 .ico 格式：https://favicon.io/
3. 将生成的 `favicon.ico` 文件放在 `public/` 目录下

## 构建生产版本

```bash
npm run build
npm start
```

## 部署到生产环境

1. 确保域名配置为：https://www.removewatermarker.com
2. 运行构建命令
3. 部署到 Vercel、Netlify 或其他平台
