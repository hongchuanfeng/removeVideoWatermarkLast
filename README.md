## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - Reusable React components
- `messages/` - Translation files for i18n
- `public/` - Static assets (favicon, images, etc.)

## Favicon

To generate a favicon.ico file:
1. Create a 32x32 or 16x16 icon image
2. Convert it to .ico format using an online tool or image editor
3. Place it in the `public/` directory as `favicon.ico`

## Build for Production

```bash
npm run build
npm start
```

## Environment Variables

No environment variables are required for basic functionality.
